使い方
「やることは、タネを選ぶ → コードを入れる → 送信 の3つだけたこ。」
1

2

「ぐらたん」か「ハーフアニバーサリー」を選びます。

コードを送信する
成功すると、このページの所持数にすぐ反映されます。

受け取ったら畑へ
受け取ったタネは、そのままたこ焼き畑で植えられます。
